# Predprof
Модуль системы наведения

Этот проект использует радиомодули FS1000A (передатчик) и MX-RM-5V (приемник) для беспроводного управления лазером и сервоприводами с помощью Arduino.

### Функциональность

* Режимы:

  * RUN H — горизонтальное сканирование.
  * RUN V — вертикальное сканирование.
  * LASER 1 — включить лазер.
  * LASER 0 — выключить лазер.

* Компоненты:

  * FS1000A (передатчик 433 МГц).
  * MX-RM-5V (приемник 433 МГц).
  * Сервоприводы для сканирования.
  * Лазер для управления.

### Подключение

* Передатчик (Arduino 1): DATA -> D3, VCC -> 5V, GND -> GND.
* Приемник (Arduino 2): DATA -> D2, VCC -> 5V, GND -> GND, Сервоприводы -> D9 и D10, Лазер -> D7.

### Установка

1. Установите библиотеки VirtualWire и ServoTimer2 через Менеджер библиотек в Arduino IDE.

### Использование

1. Загрузите код на обе Arduino.
2. Откройте Serial Monitor, чтобы вводить команды:

   * RUN H, RUN V, LASER 1, LASER 0.

### Логирование

Логирование команд и действий выводится в Serial Monitor.
